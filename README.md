# spaceship-taitanic-predictions
## Project Overview
This project presents an interactive Power BI dashboard built using the Spaceship Titanic dataset. The dashboard analyzes passenger data and visualizes predictions about whether passengers were transported to another dimension.
The goal of this project is to demonstrate data visualization, data analysis, and predictive insights using Power BI.
## 📊 Dashboard Features
The dashboard provides several interactive insights, including:
Total Number of Passengers
Total Spend Analysis
Passenger Age Distribution
Passengers by Phase (Existing Data vs New Data)
Spend by Transported Status
Predicted Transport Probability
Transported Actual vs Predicted Comparison
Model Accuracy Visualization
Users can filter data using:
Age Range
Home Planet (Earth, Europa, Mars)
Phase (Existing Data / New Data)
Transported Status
Predicted Probability
## 📈 Key Insights
The dashboard analyzes 12,970 passengers.
Total passenger spending exceeds $19M.
The prediction model achieved an accuracy of 88.31%.
Age distribution shows higher passenger concentration between 20–40 years.
